# TFM-fferretar-code
The aim of this project is to store all code and stuff necessary for TFM on Data Science Master in Universitat Oberta de Catalunya.

# Some stuff of initial configuration

* Create venv: `python -m venv tfm-env`
* Configure pycharm venv -> file > settings > python environment change interpreter from local and existing. Finally, select tfm-env
* Activate venv: `.\tfm-env\Scripts\activate.ps1`
* Installing requirements: `pip install -r requirements.txt`
* Exiting venv: `deactivate`
